Privacy Policy

Last Updated: December 2024

Important Notice: SecureHospital AI is a demonstration system containing only synthetic, fictional data. No real patient information is collected, stored, or processed.

1. Introduction

SecureHospital AI ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard information when you use our demonstration healthcare AI platform.

2. Information We Collect

2.1 Demo Account Information

When using our demo system, we may collect:

  • Username and session information
  • Interaction logs with the AI assistant
  • Access patterns for security auditing

2.2 Technical Data

We automatically collect certain technical information:

  • IP address
  • Browser type and version
  • Device information
  • Usage statistics

3. How We Use Information

We use collected information to:

  • Provide and improve the demo experience
  • Demonstrate security features and audit capabilities
  • Ensure system security and prevent abuse
  • Generate anonymized usage statistics

4. Data Security

We implement enterprise-grade security measures including:

  • Role-Based Access Control (RBAC)
  • Complete audit logging
  • Encrypted data transmission (HTTPS)
  • JWT-based authentication

5. Data Retention

Demo session data is retained for a limited period for demonstration purposes. Chat histories and audit logs may be periodically cleared.

6. Your Rights

As a demo user, you have the right to:

  • Access your session data
  • Request deletion of your demo data
  • Opt-out of analytics collection

7. Third-Party Services

Our demo may utilize:

  • OpenAI API for AI responses
  • Cloud hosting services
  • Analytics tools

8. Contact Us

For privacy-related questions, please contact the project maintainer through the project repository.

This privacy policy applies to the demonstration system only. In a production healthcare environment, additional HIPAA compliance measures would be required.
