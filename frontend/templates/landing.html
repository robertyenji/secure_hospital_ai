Role-Based Access Control

Three-Layer Security Model

Every data request passes through three independent security layers. Even if one layer fails, the others provide protection.

Layer 1: Django

Authentication, session management, and pre-flight RBAC checks before any AI interaction.

Layer 2: LLM Guardrails

System prompts instruct the AI about role limitations. AI will not attempt unauthorized operations.

Layer 3: MCP Server

Final enforcement at data layer. Validates every tool call, applies redaction, and logs to audit trail.
